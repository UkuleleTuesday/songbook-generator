Songbook Chef

Songbook Chef is an experimental songbook generator for Ukulele Tuesday. Issue reports, pull requests or improvement suggestions are welcome on GitHub.

Config-based editions

Songbook editions can be defined by YAML files found in the config/songbooks folder in this repository. To create or edit a config-based edition, add or edit a .yaml file in that directory. See the README for details on the format.

science

Google Drive editions (Experimental)

These editions are defined by a .songbook.yaml file inside a Google Drive folder. Each folder that is a direct child of the configured source folders and contains a valid .songbook.yaml will appear here. Click the folder_open icon next to an edition to open its Drive folder. Learn more.
